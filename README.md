# 🇺🇸 EN/English

> ⚠️ **Notice:** This is an updated fork of the original **[Vanilla Fireflies](https://modrinth.com/datapack/vanilla-fireflies)** datapack by **[SkyReed](https://github.com/ItzSkyReed)**. It has been updated and ported to support Minecraft 26.X

### Fireflies can be found in the swamp and mangrove swamp biome at night

## 🏞️ Supported Custom Biomes:
- Terralith
- Biomes O' Plenty
- Regions Unexplored
- Terrestria
- William Wythers' Expanded Ecosphere
- Nature's Spirit

## 📚Fireflies Settings:
- If you want to use pre-made modes use `/function vanilla_fireflies:menu`
- You can also change the amount of fireflies with `/function vanilla_fireflies:amount {amount:<an actual integer amount of fireflies particles>}`
- Or you can fine-tune the parameters using `/function vanilla_fireflies:configure {amount:<an actual integer amount of fireflies particles>, speed:<an actual float speed of particles>, delta_x:<how far along the X axis from the player can the particles appear>, delta_z:<how far along the Z axis from the player can the particles appear>, delta_y:<how far along the Y axis from the player can the particles appear>}`
  - Note: Default values are:
    - amount: 60
    - speed: 0.3
    - delta_x: 40
    - delta_y: 10
    - delta_z: 40

## 📂Installing as datapack
- Download and install datapack in the world
  - For a new world: You can click the "Data Packs" option when creating a new world and select this datapack

## 🔧Installing as mod
- Download and put mod in your .minecraft/mods folder

If you have any questions, bugs, suggestions you can [create an issue on github](https://github.com/VyrBy/Vanilla_Fireflies/issues).

### Светлячков можно найти в биомах (тут будет список) ночью.

- Terralith
- Biomes O' Plenty
- Regions Unexplored

- William Wythers' Expanded Ecosphere
- Nature's Spirit

## 📚Настройки:
- Если вы хотите использовать готовые режимы, используйте `/function vanilla_fireflies:menu`
- Вы также можете изменять количестао мсветлячков командой `/function vanilla_fireflies:amount {amount:<an actual integer amount of fireflies particles>}`
- Или вы можете точно настроить параметры с помощью `/function vanilla_fireflies:configure {amount:<количество частиц светлячков>, speed:<скорость частиц>, delta_x:<на каком расстоянии по оси X от игрока могут появиться частицы>, delta_z:<на каком расстоянии по оси Z от игрока могут появиться частицы>, delta_y:<на каком расстоянии по оси Y от игрока могут появиться частицы>}`
  - По умолчанию выставлены следующие значения:
    - amount: 60
    - speed: 0.3
    - delta_x: 40
    - delta_y: 10
    - delta_z: 40

## 📂Установка датапака
- Скачайте и установите датапак в мир
  - Для новых миров: Вы можете нажать на "Data Packs" во время создания мира и выбрать этот датапак

## 🔧Установка мода
